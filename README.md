🏥 Hospital Visitor Management System

📌 Project Overview

The Hospital Visitor Management System is a web-based application developed using PHP, MySQL, HTML, CSS, and JavaScript.
This system helps hospitals manage visitor records, patient details, and appointments efficiently.

The project runs on XAMPP (localhost) and provides an easy-to-use interface for administrators.

---

🎯 Features

- 🔐 Admin Login System
- 👤 Add Visitor Details
- 📋 View Visitor List
- 🧑‍⚕️ Patient Management
- 📅 Appointment Management
- 📊 Dashboard Statistics
- 🎨 Responsive UI with CSS Effects
- 🔎 Search and View Records

---

🛠️ Technologies Used

- Frontend: HTML, CSS, JavaScript
- Backend: PHP
- Database: MySQL
- Server: XAMPP
- Version Control: GitHub

---

💻 Installation Steps

1. Install XAMPP
2. Copy project folder to:

C:\xampp\htdocs\

3. Start Apache and MySQL in XAMPP
4. Open phpMyAdmin
5. Create a new database (example: "hospital_vms")
6. Import the provided ".sql" file
7. Open browser and run:

http://localhost/hospital_vms
